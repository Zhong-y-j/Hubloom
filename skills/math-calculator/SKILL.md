# Math Calculator

用于完成基础数学计算，并给出简短说明。

## 何时使用

- 四则运算、幂、开方、百分比
- 用户要求「算一下」「等于多少」「帮我验算」

## 步骤

1. 从用户问题中提取表达式（如 `12.5 * 8 + 3`）。
2. 运行脚本（推荐，避免心算误差）：

```bash
python scripts/calc.py "12.5 * 8 + 3"
```

3. 用一两句话解释结果；不要编造计算过程数字。
4. 非法表达式时，说明哪里不合法，并给出正确写法示例。

## 规则

- 只做数学计算，不借机写无关代码。
- 结果保留合理精度；很大/很小的数可用科学计数法。
